# Nursing AI – LM Studio Setup Guide

## Requirements
- LM Studio installed: https://lmstudio.ai
- Mistral 7B GGUF model downloaded (recommended: Q5_K_M variant)

## Instructions

1. Open LM Studio.
2. Go to the **Models** tab and download:
   `TheBloke/Mistral-7B-Instruct-v0.2.Q5_K_M.gguf`
3. After download, go to the **Chat** tab.
4. Select the model from the list.
5. In the **Custom Instructions** box (system prompt), paste the full contents of `prompt.txt`.
6. Start chatting. The model will behave as the Nursing Assistant.

## Notes
- You can save your session to avoid repeating the prompt.
- All clinical logic is offline and secure.
